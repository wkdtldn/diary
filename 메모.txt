가족들이 함께 사용하기 위한 모드 vs 자기 자신만을 위한 모드

로그인 기능을 추가하여 남이 자신의 정보를 바꾸거나 하지 못하게 한다.

역할군 처럼 가족 모드일 경우 회원 가입을 시도 할때 함께 정하도록 지정한다. 

-  회원 가입을 할때 필요한 요소
    1. 이름
    // 2. 나이
    3. 자신의 위치 (엄마, 아빠, 첫째, 둘째 ...)
    4. 성별
    5. 전화번호
    6. 생년월일
    7. 아이디
    8. 비밀번호
    9. 별명

- 로그인 할때 필요한 요소
    1. 아이디
    2. 비밀번호


- 로그인 화면에 필요한 요소
    1. input box (for ID , Password)
    2. button ( to 로그인 )
    3. button ( for 비밀번호 혹은 아이디를 잃어버린 사람 )
    4. :비어있는 input box 가 있을 시 해당 부분의 value 지우기

- 회원가입 화면에 필요한 요소
    1. input box ( for 위에 써져있는 것들 )
    2. button ( to finish ) :비여있는 input box 가 있을 시 모두 지우기
    3. button ( 뒤로가기 ) :로그인 화면으로 돌리기
